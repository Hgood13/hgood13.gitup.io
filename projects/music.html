Indie Surf Rock Music Project

Overview

This project includes songwriting, recording, and production of 30+ original indie-surf-rock tracks. With a long-term bandmate, I’ve built a signature style blending atmospheric surf tones, groove-heavy riffs, and melodic songwriting. I’m currently producing a 7-song LP with full in-house engineering.

What I Built

  • Co-wrote over 30 tracks across multiple styles in the indie/surf domain.
  • Recorded guitars, drums, bass, and vocals using a home-studio setup.
  • Engineered multitrack sessions via Behringer 18x10 interface and condenser/dynamic mics.
  • Edited and mixed tracks using DAW workflows and layered production techniques.
  • Mixed live church music for 3 years using Ableton + Behringer systems.

Key Outcomes

  • Created a cohesive sound identity for a multi-track LP.
  • Developed in-house production skills across engineering, mixing, and arrangement.
  • Built experience across both live sound and studio workflows.

Tools & Technologies

  • Behringer 18x10 Interface
  • Ableton Live
  • Condenser & Dynamic Microphones
  • Electric & Acoustic Guitars
  • Multi-mic drum recording setup
