AI Learning & Diagnostic Tooling

Overview

This project represents a structured exploration of generative AI, transformers, applied machine learning, and manufacturing diagnostic automation. The work included developing conceptual internal tools capable of making engineering diagnostics faster, clearer, and more automated.

What I Built

  • Developed structured learning frameworks for transformers, embeddings, and prompt engineering.
  • Designed data standardization and preparation pipelines for diagnostic logs.
  • Built anomaly detection concepts for vehicle bus data and event traces.
  • Prototyped AI-driven helpers for parsing CAN/LIN/Ethernet logs and triaging faults.
  • Created internal utilities to reduce manual diagnostic workload across teams.

Key Outcomes

  • Improved engineering workflow efficiency through conceptual automation tools.
  • Developed a clearer lens for how AI integrates with manufacturing engineering systems.
  • Became proficient in designing AI-enhanced diagnostic architectures.

Tools & Technologies

  • Python
  • CAN/LIN/Ethernet parsing frameworks
  • Transformers, embeddings, LLM APIs
  • Data analysis & visualization tools
