Defense & Dragroutes — AI Football RPG

Overview

Defense & Dragroutes is a two-player American-football RPG powered by 1d20 mechanics and real-time AI narration. Offense and defense players secretly select plays, the engine resolves outcomes based on system tables, and the game narrates results using dynamic prompts to the ChatGPT API. I built the web UI, game logic, and the narration framework.

What I Built

  • Designed the 1d20 outcome system, play tables, coverage shells, and state machine.
  • Built the full HTML/CSS/JS front-end with a 4-panel grid interface.
  • Implemented API calls for live AI narration based on play selections.
  • Created hidden play-call mechanics so opponents cannot see choices.
  • Developed formation previews, defensive disguises, and buffered outcomes.
  • Prototyped gameplay loops and session logic for repeated drives and quarters.

Key Outcomes

  • Created a unique hybrid between tabletop RPG logic and football strategy.
  • Developed reusable game-state structures for future game expansions.
  • Delivered a functional local multiplayer experience over a simple web UI.

Tools & Technologies

  • JavaScript (game logic)
  • HTML/CSS (grid layout)
  • ChatGPT API
  • Dice-based probability modeling
