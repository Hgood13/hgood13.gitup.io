Progressive Clue Guessing Game

Overview

This project is a React-based logic game inspired by Wordle and Connections. Players guess an object using progressively clearer clues. The game includes dynamic state management, clue iteration, scoring, animations, and a mobile-responsive UI with a Wordle-inspired aesthetic.

What I Built

  • Developed the entire React front-end using functional components and Hooks.
  • Built the progressive clue engine with adjustable difficulty.
  • Implemented guess validation, history tracking, and per-round scoring.
  • Designed a sleek UI with emoji feedback, animations, and dark-theme elements.
  • Created responsive layouts for mobile and desktop play.
  • Iteratively tested difficulty balance and user experience flows.

Key Outcomes

  • Built a complete mini-game with replayability and difficulty tuning.
  • Learned deeper React patterns and state synchronization concepts.
  • Created a foundation for a larger, multi-round puzzle experience.

Tools & Technologies

  • React (Hooks, state, effects)
  • CSS animations and UI styling
  • JavaScript logic engines
