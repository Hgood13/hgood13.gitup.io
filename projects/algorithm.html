The Algorithm — Social Persona Card Game

Overview

The Algorithm is a party game concept where players respond to prompts using fictional online personas. Characters mimic social-media archetypes, complete with bios, hashtags, search histories, and algorithm recommendations. Players improvise responses using dry-erase cards, creating replayable, personality-driven fun.

What I Built

  • Designed the entire game concept and persona system.
  • Created multiple character profiles (GamerGavin, MamaVibesMaria, etc.).
  • Art-directed and generated modern, neon-accented card visuals.
  • Built a card structure with dry-erase answer cards for live play.
  • Designed prompt mechanics using “algorithmic recommendation” humor.

Key Outcomes

  • Developed a playable party game with strong creative identity.
  • Created a visual and thematic style matching modern digital culture.
  • Generated reusable persona templates for rapid expansion.

Tools & Technologies

  • AI Art Generation Tools
  • Graphic Design Layouts
  • Persona & Prompt Design
  • Paper prototyping with dry-erase components
